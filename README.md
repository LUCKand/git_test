*   2020.11.05

    学会了使用ssh连接远程git服务器，gitee上面的引导非常好  
    本地先配置[SSH key](https://gitee.com/help/articles/4181#article-header0)  

        ssh-keygen -t rsa -C "xxxxx@xxxxx.com" 
        cat ~/.ssh/id_rsa.pub
        ssh -T git@gitee.com

    然后再再本地的仓库添加远程git服务器，遇到问题百度。推送到github就不用输入密码了。
